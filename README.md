# Introduction

<굳세어라 간조메> 는 슈팅게임이다. Mangoboard64는 LCD화면, 터치 스크린, graphic processor 등을 내장하고 있으며 주어진 시스템을 이용해 만들 수 있는 가장 가치있는 것이 슈팅게임일 것이라고 판단하여 제작하게 되었다.
